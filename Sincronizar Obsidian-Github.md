Hoy aprendí a sincronizar y llevar los avances y apuntes de mi Obsidian en Github también para tener un perfil mas completo mostrando mi avance en cuanto a mi aprendizaje.

Me resulto algo complejo en la configuracion del plug in en Obsidian pero al fin se pudo iniciar sesion en Github con Git 

Los comandos basicos en Windows shell para subir actualizaciones de Obsidian son los siguientes:

[cd] que es changue directory, sirve para moverte entre carpetas, como cuando navegas en el explorador de archivos pero con texto.

[git add .]  con este comando vas a agregar lo cambios hechos en esa carpeta

[git commit -m] con este comando vas a agregar una nota y guarda los cambios, es un paso necesario, no es opcional.

y por ultimo usas el comando 

[git push] Toma todos los commits guardados localmente y los sube a Github.

esos son los 4 indispensables para subir todo lo que agregues a Obsidian a Github. Existen mas comandos que tambien son muy utiles como:

[git pull] para descargar lo que este en Github muy util si trabajas en 2 equipos diferentes.

[git --version] Ver version instalada, solo para verificar que Git está instalado correctamente.

[git status] Ver estado actual, Te muestra qué archivos han cambiado, cuáles son nuevos y cuáles están listos para subir.

[git clone] Clonar repositorio, descarga una copia completa de un repositorio de GitHub a tu computadora.
