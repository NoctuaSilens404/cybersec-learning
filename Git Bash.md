## ¿Qué es Git Bash?

Git Bash es una terminal que viene incluida cuando instalas Git en Windows. La palabra **Bash** viene de **Bourne Again Shell**, que es el intérprete de comandos más usado en Linux.

Básicamente es una **terminal de Linux funcionando dentro de Windows**.

---

## ¿Por qué es mejor para aprender que PowerShell o CMD?

|                                 | Git Bash                 | Powershell      | CMD              |
| ------------------------------- | ------------------------ | --------------- | ---------------- |
| Comandos                        | Linux(ls, cd, cat, grep) | Windows propios | Windows antiguos |
| Uso de ciberseguridad           | Si                       | Poco            | Casi nada        |
| Uso en servidores               | La mayoria son Linux     | No              | No               |
| Compatibilidad con herramientas | Total                    | Parcial         | Mínima           |

---

## La razón más importante

El **99% de los servidores del mundo corren Linux**. Cuando trabajes en ciberseguridad estarás atacando o defendiendo sistemas Linux constantemente. Cada comando que aprendes en Git Bash es exactamente el mismo que usarás en:

- Servidores reales
- Kali Linux (sistema operativo de hacking)
- Hack The Box y TryHackMe
- Tu proyectos cuando los despliegues en un servidor

Entonces cada vez que abres Git Bash no solo estás usando Git, estás practicando Linux sin darte cuenta. Es matar dos pájaros de un tiro.
